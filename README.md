# Stranger Things Fan Page

## Descripción del Proyecto
Fan page dedicada a la serie "Stranger Things" de Netflix, desarrollada con HTML y CSS puro siguiendo las especificaciones de la consigna académica.

## Estructura del Sitio

### 1. Página Principal
- Presentación de la serie con información general
- Diseño hero con imagen y call-to-action
- Navegación fija con efectos hover

### 2. Sección de Personajes
- 6 personajes principales con imágenes y descripciones
- Cards con efectos hover y animaciones
- Grid responsive que se adapta a diferentes tamaños de pantalla

### 3. Sección de Temporadas
- Información de las 3 primeras temporadas
- Listado de episodios destacados
- Cards numeradas con efectos visuales

### 4. Formulario de Contacto
- Campos para nombre, email, personaje favorito y comentarios
- Checkbox para suscripción a newsletter
- Validación HTML5 y estilos personalizados

## Características Técnicas

### Tipografías
- **Creepster**: Para títulos y elementos principales (estética de terror/ciencia ficción)
- **Roboto**: Para párrafos y texto general (legibilidad y modernidad)

### Paleta de Colores
- **Primario**: #ff6b6b (rojo característico de la serie)
- **Secundario**: #ff8e53 (naranja complementario)
- **Fondo**: #0a0a0a (negro profundo)
- **Texto**: #e0e0e0 (gris claro)
- **Footer**: Gradiente marrón-morado (#2c1810 a #0f0a1a)

### Efectos y Animaciones
- Efectos hover en navegación, imágenes y botones
- Transformaciones 3D en imágenes de personajes
- Sombras con glow effect usando el color de la marca
- Transiciones suaves en todos los elementos interactivos
- Botones con gradientes y efectos de elevación

### Características Responsivas
- Grid layout que se adapta automáticamente
- Breakpoints para tablet (768px) y móvil (480px)
- Navegación que se reorganiza en pantallas pequeñas
- Imágenes optimizadas para diferentes tamaños

## Tecnologías Utilizadas
- HTML5 semántico
- CSS3 con Flexbox y Grid
- Google Fonts (Creepster y Roboto)
- Unsplash para imágenes de placeholder

## Funcionalidades Destacadas
- Navegación suave entre secciones
- Formulario funcional con diferentes tipos de input
- Diseño completamente responsive
- Efectos visuales acordes a la temática de la serie
- Footer con color distintivo según requisitos

## Instrucciones de Visualización
1. Abrir el archivo `index.html` en cualquier navegador web moderno
2. La página cargará automáticamente las fuentes de Google Fonts
3. Navegar usando el menú superior o scroll natural

## Compatibilidad
- Navegadores modernos (Chrome, Firefox, Safari, Edge)
- Dispositivos móviles y tablets
- Resoluciones desde 320px hasta 1920px+
